# Checklist de entrega

Antes de entregar tu documentación, confirma:

- [ ] El sitio compila en GitHub Pages.
- [ ] El menú lateral tiene orden lógico.
- [ ] Cada página tiene `title` y `nav_order`.
- [ ] Las imágenes cargan (rutas relativas correctas).
- [ ] Los bloques de código tienen lenguaje (cuando aplica).
- [ ] Los links internos no están rotos.
- [ ] Incluiste referencias/citas si tomaste material externo.

Opcional:

- [ ] Incluiste un video demo.
- [ ] Incluiste un PDF de referencia (datasheet, artículo, manual).
- [ ] Incluiste una sección de “créditos” y licencia.
